# PlayTime
Проект, разработанный для общения между людьми при помощи микрофона (аналог Discord)

На данный момент нет ни меню, ни готового решения.
Причина: Очень проблемный баг, который никак не получается решить - присоединение больше одного человека в разговор с одним лицом.

Что имеется ввиду?
- Когда в приложение входит больше чем два человека, воспроизведение происходит очень криво. Подразумеваю, что проблема клиентская, метода к решению нет, было потрачено очень много времени.
